About E-Quiz

  • It is a quiz portal with timer for each quiz
  • User can login using their google account
  • User can download their report card at the end of Quiz
  • Once an option for the question is clicked, it can't be changed later
  • User can download their report card at the end of Quiz
  • User can't copy the question

Special features

  • Admin can add and delete quizes at anytime
  • At the end of each quiz, user can view the review page where correct answer for that particular question is displayed
  • Only admin has access to view the user's activity
